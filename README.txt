HƯỚNG DẪN CHẠY ĐỒ ÁN QUẢN LÝ KHO

1. Chạy Backend (Server):

Mở Terminal mới, gõ: cd backend

Gõ tiếp: npm install 

Gõ tiếp: npm start

2. Chạy Frontend (Giao diện):

Mở Terminal mới thứ 2, gõ: cd frontend

Gõ tiếp: npm install

Gõ tiếp: npm run dev
